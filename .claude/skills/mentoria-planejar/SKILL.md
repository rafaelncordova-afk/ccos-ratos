# Skill: mentoria-planejar

Apoia as mentorias voluntárias de planejamento financeiro do Rafa como CFP® nos programas sociais da Planejar (ex: Boogie Week). Ajuda a registrar encontros, gerar mensagens de resumo para o mentorado e planejar a pauta dos próximos encontros com foco em resultado.

## Quando usar

- Após um encontro: para registrar o que foi conversado e gerar mensagem de acompanhamento
- Antes de um encontro: para revisar o histórico e sugerir uma pauta focada
- Quando o usuário chamar `/mentoria-planejar` ou pedir "registra o encontro", "prepara a pauta", "o que abordamos com [nome]"

## Fontes de dados

Para cada mentorado, ler o arquivo:
`educação financeira/planejar/[programa]/[Nome do Mentorado]/encontros.md`

Esse arquivo contém: perfil consolidado, histórico de todos os encontros, mensagens enviadas e tarefas em aberto.

## Modo 1 — Pós-encontro (registrar + mensagem)

**Quando usar:** logo após uma sessão, quando o Rafa passa o que foi conversado.

**Fluxo:**
1. Perguntar (se não souber): qual mentorado, qual número do encontro, data
2. Ler o `encontros.md` do mentorado para ter contexto do histórico
3. Registrar o novo encontro no arquivo, no campo correspondente, com:
   - Temas abordados (narrativa fluida, não bullet list seca)
   - Tarefas combinadas para o próximo encontro
4. Gerar mensagem de acompanhamento para envio ao mentorado (ver regras abaixo)
5. Salvar a mensagem final no arquivo, após confirmação do Rafa

**Regras da mensagem de acompanhamento:**
- Tom: próximo, humano, sem formalidade excessiva — como alguém que se importa de verdade
- Linguagem: simples, acessível, sem jargão técnico
- Estrutura: breve resumo do que foi conversado + tarefas claras numeradas + data dos próximos encontros + assinatura
- Assinatura: Rafael Córdova, CFP®
- Nunca usar travessão (—)
- Sem frases genéricas ("espero que esteja bem", "fico à disposição")
- O texto deve soar como se o Rafa tivesse escrito, não como IA

## Modo 2 — Pré-encontro (planejar pauta)

**Quando usar:** antes de uma sessão, para o Rafa chegar preparado.

**Fluxo:**
1. Ler o `encontros.md` completo do mentorado
2. Identificar:
   - Tarefas que o mentorado ficou de fazer — foram cumpridas? Precisam ser retomadas?
   - Temas abordados anteriormente que ficaram incompletos ou geraram dúvidas
   - Qual é o momento da jornada (1º, 2º encontro etc.) e o que faz sentido aprofundar agora
   - Gaps no perfil financeiro que ainda não foram explorados
3. Sugerir pauta estruturada para o próximo encontro (ver progressão abaixo)
4. Destacar perguntas-chave que o Rafa pode fazer para aprofundar o diagnóstico ou avançar

## Progressão típica das mentorias

Usar como referência para sugerir pautas — adaptar sempre ao que o mentorado traz:

| Encontro | Foco sugerido |
|---|---|
| 1 | Apresentação, escuta ativa, diagnóstico inicial, vínculo de confiança |
| 2 | Aprofundar as dores centrais levantadas no diagnóstico; retomar tarefas |
| 3 | Separação financeira pessoal/negócio (se aplicável); mapeamento de receitas e gastos |
| 4 | Situação das dívidas: entender o quadro completo e montar estratégia de priorização |
| 5 | Construção de hábito: fluxo de caixa simples, orçamento básico, reserva de emergência |
| 6 | Revisão de evolução, celebrar avanços, próximos passos após o programa |

## Princípios do CFP® voluntário (guia de postura)

Sempre considerar ao sugerir pautas e abordagens:

**Fazer:**
- Escuta ativa e empatia
- Linguagem simples e respeitosa
- Orientar ações práticas e possíveis dentro da realidade do mentorado
- Identificar uma dor central por encontro — não tentar resolver tudo de uma vez

**Não fazer:**
- Indicar produtos, instituições financeiras ou serviços específicos
- Promover-se comercialmente
- Julgar decisões passadas
- Pré-julgar pelo histórico ou classe social
- Pressa em "resolver" — o processo é gradual

## Ao final do ciclo

Lembrar o Rafa de:
- Pedir ao mentorado para preencher o formulário final de evolução (QR Code do guia)
- Registrar as horas voluntárias (QR Code do guia)
