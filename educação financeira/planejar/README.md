# Planejar — Programas voluntários

Rafa é porta-voz da Planejar e atua como voluntário CFP® em programas de educação financeira.
Cada programa tem sua própria subpasta aqui dentro.

**Estrutura:**
- `planejar/[nome do programa]/` — uma pasta por programa (ex: `boogie week 2026/`)
  - `[Nome do assistido]/encontros.md` — histórico de encontros com cada pessoa atendida
  - `materiais/` — materiais de apoio usados no programa

Ao começar um novo programa voluntário, criar uma nova subpasta seguindo esse mesmo padrão.
